# Swarm Persona Index

Use this compact index before opening `personas.md`. Select the smallest set that covers the work.

| Persona | Use When |
| --- | --- |
| Researcher | Need independent evidence gathering or source comparison |
| Spec Writer | Need requirements, acceptance criteria, or BDD scenarios |
| Architect | Need design alternatives, tradeoffs, or system boundaries |
| Implementer | Need patch proposals or execution planning |
| Reviewer | Need adversarial quality checks or risk assessment |
| Security | Need threat modeling, secrets, auth, data, infra, or dependency review |
| Verifier | Need test strategy, proof criteria, or reproducible validation |

Open the full `personas.md` only when this table is too coarse for the requested swarm.
