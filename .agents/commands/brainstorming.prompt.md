## Setup
You are facilitating an interactive, multi-turn brainstorming session.

To ensure the session is structured, premium, and highly productive, **load and adhere to the `brainstorming` skill**.

## Interactive Facilitation Flow

1. **Frame the Problem**: Guide the user through **Phase 1: Frame the Problem** of the `brainstorming` skill. Ask probing questions to refine their statement.
2. **Select Technique**: Based on the problem type, recommend 1 or 2 specific techniques from **Phase 2: Divergent Techniques**. Let the user choose or select a custom approach.
3. **Ideate**: Run fast, judgment-free generation rounds following the **Facilitator Rules** in **Phase 3**. Prompt the user to build on ideas ("Yes, and...").
4. **Group & Score**: Lead the user through **Phase 4: Convergent Techniques** to cluster ideas and score them on Impact, Effort, and Risk.
5. **Evaluate**: Challenge the top 3-5 ideas using the **Phase 5: Multi-Perspective Evaluation** lenses (Product, Architecture, Security, Devil's Advocate).
6. **Action Plan**: Compile the final results into a structured Markdown document using the template in **Phase 6: Action Plan & Output**.
